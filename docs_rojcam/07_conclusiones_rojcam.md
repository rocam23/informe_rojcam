Para concluir, se puede decir que existen varias medidas para mantener un sistema al día cuando se trata de prepararse ante amenazas informáticas. Principalmente, mantener el software y manejo de seguridad al día, tal como usar un cifrado moderno y seguro para tratar datos sensibles. Esto es una parte importante de este caso, ya que uno de los problemas en la brecha de Yahoo fue que varios datos usaban el cifrado MD5, que es considerado fácil de interceptar hoy en día.  
Además, capacitar a personal y posiblemente usuarios para que manejen medidas de protección, principalmente no usar la misma contraseña para varias credenciales y reconocer ataques de tipo phishing antes de que estos puedan perjudicarlos.
